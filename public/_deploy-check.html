2 마진율 갭 시각화

📈
신제품 폼 — 마진율 컬럼 시각 점검 (수동)
신제품 폼 진입 → 8국 표 가장 우측에 "마진율 (한국→수출)" 컬럼 표시 확인
수동 점검 절차:
① 신제품 사업성 검토 새 탭으로 열기 → Cmd+Shift+R 강제 새로고침
② 임의 프로젝트 선택 (또는 새로 입력: 제품명·품목·생산단가·수량·타겟가)
③ 결과 표 우측 끝 컬럼: 한국 X% / → Y% / 갭 ±Z% 표시 확인
④ "확정 현지가" input 에 권장값보다 낮은 값 입력 → 갭 빨강(-N%) 변화 확인
⑤ 권장값 그대로 → 갭 0 근처 (라운딩 차이만)
수동
📑
preview-recalc Excel — 4개 신규 컬럼
FOB원 / 한국마진pct / 수출후마진pct / 마진갭pct (FOB 없으면 N/A)
수동 점검 절차:
관리자 환경변수 ADMIN_PASSWORD 알면 직접 다운로드 → /api/admin/pricing-audit/preview-recalc-2026-04-28?password=XXX
또는 미설정 시 빈 password 통과 → 파일 다운로드 → 우측 4개 컬럼 + 색상 코딩 확인
대기
